Power BI workbook already has the data imported and user should be able to interact with the dashboards. However, if you would like to refresh data please follow the below steps

1) Import the data into SQL server database (name it - Training).
2) Name the table as RData ('dbo' schema)
3) Connect the Power BI workbook to the SQL server database and refresh. 

Query used for importing data is available in CSQuery.sql file
